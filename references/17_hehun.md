# 合婚 · 把两张命盘并起来看相处模式与磨合点

> 这是**合婚落地层**，承接夫妻星宫（`04`）、旺衰用神（`02`）、刑冲合会（`01`）、行运同看（`14`）。前几步把双方各自的命盘读透，这一步把两张盘并到一起，看两个人放在一段长期关系里，能量上哪里互补、哪里相磨，给出可执行的相处建议。
>
> 三条铁律：
> 1. **禁打分制、禁二元判词**：不给「合婚分」「契合度百分比」，不下「合」「不合」「天作之合」「八字不配」这类一刀切结论。只输出相处模式与具体磨合点，把判断权交还给当事人两个人。
> 2. **组件有典、流程为今人整理**：合婚拆开看，每个组件都有古法依据，夫妻星宫出《三命通会》六亲与宫位、用神互补承《滴天髓》中和、日柱与年支的合冲走刑冲合会（`01`）；但「把两张盘并起来逐项过」的综合流程是今人为落地整理的工程化次第，属于今人编排，古籍里没有这套原装章法，引用时如实标注。
> 3. **祛魅，不吓人**：不做属相相冲一票否决（如「龙虎相斗」「鸡狗不到头」），不做合婚煞法（如生年一字配出大忌的旧合婚书）。这类做法《命理约言》早有批判，见本篇第五节。

## 一、合婚先看什么：先各自看清，再并起来

合婚最常见的误区是「拿两个生年属相一对就下结论」。**正确的次第是：双方各自先走完旺衰用神（`02`）、夫妻星宫（`04`），再把两张盘并到一起逐项对看。** 各自没读清，合在一起只会以讹传讹。

- **看的是关系里的能量配比**：一方旺的地方另一方是否需要，一方缺的地方另一方能否补，两个人的命门字凑到一起是相互补台还是相互拆台。
- **看的是相处模式与磨合点**：合婚只描出两个人放在一起最舒服的相处方式，以及最容易起摩擦的那几个点，提前知道、提前经营，止步于此，不去预测这段关系成或败、分或合。
- **决定权在两个人**：命理只提供一个参考维度，是否在一起、怎么过，由当事人结合现实感情理性拍板。

## 二、合婚方法五步

按次第逐步过，每一步都锚到双方的具体命盘，不套通用话术。

### 第一步 · 双方各自的婚姻象

先把两人各自的婚姻底子读出来，这是合婚的地基（承 `04` 婚姻·感情四看）。

- **夫妻星旺衰**：男命看财星（妻）、女命看官星（夫）的旺衰喜忌。夫妻星为喜用且得力 → 对配偶倾向重视、关系有滋养；夫妻星为忌或受伤 → 在亲密关系里易有功课，需后天经营。
- **夫妻宫（日支）状态**：日支为喜用且不受刑冲 → 婚姻宫安稳、家宅倾向和顺；日支为忌或逢冲刑 → 关系易起波动，需留意。
- 两人各自的婚姻象先各自成立，再看放到一起会怎样相互影响。

### 第二步 · 用神互补判定（承 `02` 中和）

这是合婚的核心一步，承《滴天髓》「命局以中和为贵」。两个人在一起，本质是两套五行能量的叠加，看叠加后是趋向中和还是加剧偏枯。

- **加分项（互补）**：一方的用神，恰是另一方的喜用五行。例：甲方身旺极、用神为水（需要泄耗降温），乙方命中水旺且为乙方喜用，这样两人在一起，乙方的水能给甲方降火，能量上相互补台，相处倾向越处越顺、相互成全。
- **磨合点（相拗）**：一方的用神，恰是另一方的忌神。例：甲方用火暖局，乙方却最忌火（身旺火炎），这样甲方靠近的能量正是乙方要回避的，长期相处易在节奏、脾性、生活方式上相互掣肘，需要后天有意识地各退一步、错开锋面。
- 互补与相拗常常同时存在，一对组合里往往既有补台处也有相磨处。**如实把两边都列出来，不合并成一个总分。**

### 第三步 · 日柱与年支的干支关系（承 `01` 刑冲合会）

把双方的日柱、年支放在一起，看天干地支的合冲刑害，这是可计算的关系事实层（依据 `01` 六、七节）。

- **天合地合**（如甲子配己丑，天干甲己合、地支子丑合）：相互吸引、牵绊深，缘分感强，但合多易黏腻、易相互迁就到失去边界，需留意各自独立空间。
- **天克地冲**（如甲子配庚午，天干甲庚冲、地支子午冲）：相互的张力大、火花足，也最易正面碰撞，相处需要更多缓冲与沟通。
- **六合·三合**（日支或年支构成子丑合、寅午半合等）：地支相牵，倾向亲近、好商量。
- **相刑·相害**（构成寅巳、子未等刑害）：暗处易有嫌隙磨损，多为细水长流的小摩擦，靠日常经营化解。
- 这一层只给「干支之间是什么关系」这个事实，关系强弱与好坏的解读，仍要回到第二步用神互补一起看，不单凭一个合或冲下判断。

### 第四步 · 关键大运同步性（承 `14` 行运同看）

承《子平真诠》「论运与看命无二法」。合婚除了看原局，还要看两个人未来十年的运势节奏是否合拍。

- **按公历年对齐**：把双方各自的大运排出来，以同一个公历年为基准对齐（双方起运岁数不同，须按公历年份对齐，不按岁数对齐），看同一年两个人各走什么运。
- **看同步性**：同一段时间里，双方是各自都走助用神的好运（节奏同向上行、相互托举），还是一方走助用、另一方引忌神（一个往上、一个往下，节奏错位易相互拖累或不理解），还是双方同时走引忌的运（共同的低谷期，需相互扶持守过）。
- 大运同步性看的是**节奏合不合拍**，止于趋势，不预测某年必有某事。两人节奏长期错位的，仍可以过下去，这一条只提示在对方低谷期多一份体谅。

### 第五步 · 相处建议

把前四步的结论收口成可执行的相处建议，正反成对地给。

- **互补处放大**：用神相互补台的地方，鼓励发挥（如一方稳一方活、一方主外一方主内，能量上正好接得住），点出来让两人知道这是天然顺手处。
- **磨合点经营**：用神相拗、日柱相冲、大运错位的地方，给具体低门槛的相处方法（错开锋面、提前沟通、低谷期互撑、给彼此独立空间），把「磨合点」翻译成「这件事上多留个心」。
- 全程用「倾向」「易」「利于」「需注意」的趋势化措辞，不写「你们必然如何」「这段关系会怎样」。

## 三、脚本支撑：一次排两盘、打印双盘交互表

排盘交给脚本精确算，方法论推演由人来做。命令：

```
python3 scripts/paipan.py <甲方生辰> --gender X --partner <乙方生辰> --partner-gender Y
```

脚本会一次排出双方两张完整命盘，并打印一张**双盘交互表**，给出可计算的关系事实：

- 双方日柱的天干地支关系（天合地合 / 天克地冲 / 比和等）；
- 双方年支、日支之间的六合 / 三合 / 六冲 / 相刑 / 相害；
- 夫妻宫（日支）相互之间有无刑冲合害；
- 双方五行个数的互补情况（一方多的、另一方少的五行）；
- 生肖（年支）的合冲关系，仅作事实呈现。

**交互表只给可计算的关系事实，不打分、不下结论。** 用神互补判定（第二步）、大运同步性解读（第四步）、相处建议（第五步）这些需要旺衰喜忌判断的部分，仍由本篇方法论逐步推演，脚本不代劳、更不给契合分。五行个数互补只是粗看，是否真互补要回到「一方用神是否为另一方喜用」来定。

## 四、时辰缺失的降级规则

合婚常遇到一方或双方报不准时辰。缺时辰仍能合，做法是分层降级：能做的照做，做不了的明确标存疑，不硬断。

| 项目 | 缺时辰时能否做 | 处理 |
|---|---|---|
| 日柱天干地支关系 | 能 | 日柱由日期定，与时辰无关，正常推 |
| 夫妻星旺衰、用神互补 | 多数能 | 旺衰用神主要看月令、日主、年月日三柱，缺时柱仍可判主轴，结论略留余地 |
| 年支 / 日支六合六冲 | 能 | 与时支无关，正常推 |
| 时支六合六冲 | 否 | 时支未知，凡涉及时支的合冲一律标「存疑·待时辰确认」 |
| 子女宫（时柱）相关 | 否 | 时柱为子女宫，缺则不断子女缘，标存疑 |
| 晚运大运同步性 | 部分 | 缺时辰则起运与大运排布存在不确定，中晚年那几步标存疑，早中年相对可靠 |

> 一句话：**日柱关系、夫妻星旺衰、用神互补在缺时辰时仍可做主轴判断；时支六合六冲、子女宫、晚运同步必须标存疑，不拿不确定的字下结论。**

## 五、祛魅：不做属相一票否决，不做合婚煞法

合婚是民间命理被附会得最重的一块，旧俗里大量「属相相冲不能婚」「年命犯某煞大忌」的吓人说法，缺生克正理。本篇明确不做这些。

- **不做属相相冲一票否决**：只凭两人生肖（年支）相冲、相刑就断「不能在一起」，是只看一个字定终身，舍弃了月日时与用神的全盘信息。《命理约言》直指「以人之生年支生肖所属，论人吉凶，尤为谬妄」（`10` 第六节生肖论命）。生肖合冲在本篇只作第三步里一个可计算的事实，参考权重低，绝不单独定论。
- **不做合婚煞法**：旧合婚书「止以年一字，犯某日某时为言」，仅凭生年一个字配出大忌来吓唬人，缺生克正理，《命理约言》将其列入祛魅清单（`10` 第六节合婚煞法）。本篇取代它的做法，是回到双方完整命盘的用神生克与中和来看相处，把判断建立在可解释的能量配比上。
- **回到生克正理**：合婚的依据是两套五行能量叠加后趋向中和还是偏枯、双方用神是否互补、大运节奏是否合拍，全部讲得出生克道理，不靠神煞名目与口诀歌赋。

## 六、双盘脱敏命例（骨架演示）

用两个虚构生辰走一遍五步流程，演示「用神互补 + 一处磨合点」怎么落地。生辰均为虚构脱敏，仅供方法论研究。

**甲方**（虚构）：丁火日主，生于午月，身旺极、满盘火炎缺水，用神为水（调候降温），喜金（生水），忌火、仇木。
**乙方**（虚构）：壬水日主，生于子月，水旺，用神为火土（调候暖局、止水），喜木（生火），忌金水。

- **第一步 · 各自婚姻象**：甲方男命以财（金）为妻，日坐酉金财星得地，夫妻星有根、与配偶倾向有缘。乙方女命以官（土）为夫，官星藏而得用，对伴侣倾向有期待、重名分。两人各自的婚姻底子都不算薄。
- **第二步 · 用神互补**：甲方用神为水，乙方一身水旺，乙方的水正好能给甲方过旺的火降温，是明显的**加分互补项**（一方所需正是另一方所有）。反过来看，甲方满盘火炎，而火恰是乙方的喜用（乙方水旺需火暖局），甲方的火能暖乙方的局，**双向互补**，能量上相互补台。
- **第三步 · 日柱年支关系**：设甲方日柱丁酉、乙方日柱壬午，天干丁壬合（合化木）、地支酉午相破。天干相合主相互吸引、缘分牵绊深；地支一处相破，是**一处磨合点**，提示两人在某些生活节奏上易有细小摩擦，靠日常沟通化解。
- **第四步 · 大运同步性**：按公历年对齐，若某十年甲方走金水助用的好运、乙方同期走木火助用的好运，两人节奏同向上行、相互托举，是顺的同步期；若某段甲方走火运（引忌）、乙方走金水运（引忌），双方同时偏弱，则为共同低谷，需相互扶持。具体年份代入双方各自大运逐年看。
- **第五步 · 相处建议**：互补处放大，两人能量上水火相济、相互降火暖局，是天然顺手的搭配，鼓励发挥一稳一活、相互调节的相处模式。磨合点经营，日支相破的小摩擦，提示在生活习惯、节奏安排上多留心、提前沟通；大运共同低谷期相互扶持、不相互埋怨。

> 全程演示只描相处模式与磨合点，**没有给契合分、没有下「合不合」的结论**，断语全部趋势化。当事人据此参考，最终自行拍板。
