# 六爻（纳甲筮法）· 装卦与用神断事

> 这是本库第三个引擎：**六爻占卜**。与梅花易数同属易卦占事（占一件具体的事），比梅花多一整套装卦系统（纳甲干支、八宫世应、六亲、六神），断事更细、信息量更大。装卦交给 `scripts/liuyao.py` 装准，本篇管推演（取用神、旺衰、动静生克、应期）。
>
> 三条铁律：
> 1. **占卜非命理**：一事一占，问什么断什么，不批一生。同一件事不反复摇卦。
> 2. **趋势化，不打分、不铁口**：断成败只说易成 / 有阻 / 需注意，落到可执行提醒，不下「必成必败」判词。
> 3. **装卦靠脚本、断卦靠方法论**：纳甲、世应、六亲、六神是最容易装错的机械层，交脚本；取用神与生克旺衰的判断靠本篇。

## 一、摇卦与装卦

**摇卦**：三枚铜钱摇六次，自初爻（最下）向上记：3 背 = 老阳 9（动）、2 背 = 少阳 7、1 背 = 少阴 8、0 背 = 老阴 6（动）。老阳老阴为动爻，动则变（阳变阴、阴变阳），动爻可多个（与梅花单动爻不同）。

**装卦五件套**（脚本 `--yao 六位数 --date 年 月 日` 一次装全）：

1. **纳甲**：每爻配干支。天干按京房纳甲（乾纳甲壬、坤纳乙癸、震庚、巽辛、坎戊、离己、艮丙、兑丁），地支按各经卦纳支表自初爻向上。
2. **定宫**：六十四卦分属八宫（本宫首卦→一至五世→游魂→归魂），宫卦五行即「我」。
3. **世应**：纯卦世六，一至五世卦世一至五，游魂世四、归魂世三；应爻与世相隔三位。世为自己、应为他人他方。
4. **六亲**：以宫五行为我，生我者父母、我生者子孙、克我者官鬼、我克者妻财、比和者兄弟，逐爻按纳支五行配。
5. **六神**：按起卦**日干**起：甲乙起青龙、丙丁朱雀、戊勾陈、己螣蛇、庚辛白虎、壬癸玄武，自初爻向上循环。

另取起卦日的**月建**（节气月支）、**日辰**（日干支）、**旬空**，为旺衰与应期之用。

## 二、取用神（断事第一步）

测什么事，取对应六亲为**用神**；生用神者为**元神**，克用神者为**忌神**，生忌神者为**仇神**：

| 所测之事 | 用神 |
|---|---|
| 父母、长辈、文书、房屋、车船、消息 | 父母爻 |
| 官运、官司、丈夫（女占）、疾病之因 | 官鬼爻 |
| 财运、买卖、妻子（男占）、财物 | 妻财爻 |
| 子女、晚辈、医药、解忧、宠物 | 子孙爻 |
| 兄弟朋友、同辈、竞争者、破财之源 | 兄弟爻 |
| 自身吉凶 | 世爻 |
| 对方、对手、合作方 | 应爻 |

用神不上卦（六爻中无该六亲）时查**伏神**（本宫首卦该位六亲伏于飞爻之下），伏神旺相可出则事有转机、伏而受制则难。

## 三、旺衰与动静生克

**旺衰看月建日辰**：用神得月建生扶（当令、相）为旺，逢月破（被月建冲）大凶象；得日辰生合为有力，被日辰冲克为受制。旬空之爻「空亡」，事多悬而未决，出空（过旬）应事。

**动静**：静爻安、动爻事。动爻能生克他爻，静爻受生克而不主动作用。动爻变出之爻回头生本爻为**回头生**（吉助），回头克为**回头克**（凶象加重）。

**断事总纲**（趋势化）：
- 用神旺相、动而化吉、得元神生扶：事**易成、倾向顺遂**。
- 用神休囚、被忌神克制、逢空破：事**有阻、需注意**，待应期或另图。
- 世应相生相合：双方**易合作**；世应相冲相克：**多分歧**，宜先协调。
- 六神辅断色彩：青龙主喜庆、朱雀主口舌文书、勾陈主田土迟滞、螣蛇主怪异纠缠、白虎主凶伤疾病、玄武主暗昧盗失。六神只作象义参考，不独立定吉凶。

## 四、应期

应期常取：用神逢**值**（当值之日月）、逢**合**、逢**冲**（冲开合绊、冲实空亡）之时；空亡看**出空**之日；月破看**出月填实**。只给可参考的时间段，不铁口定日。

## 五、脚本用法

```bash
python3 scripts/liuyao.py --yao 787888 --date 2026 6 15 --query "问合作"
# --yao：摇卦结果六位（自初爻向上，6=老阴动 7=少阳 8=少阴 9=老阳动）
# --gua 上卦数 下卦数 动爻：直接指定（0=静卦），卦数用先天八卦数
# --date：起卦日期，装月建/日辰/旬空/六神；不传则略过；恰逢交节日补第 4 个数（时）精确定月建
```

脚本输出：本卦（卦名 / 宫 / 世应）、逐爻六神 + 六亲 + 纳甲干支五行 + 爻画 + 动爻标记与变出之爻、变卦名、月建日辰旬空。装卦是机械定式全交脚本，断事按本篇二至四节推。

## 六、装卦范例（脱敏演示）

**所占**：某人问一桩合作能否谈成。摇得 `787888`（二爻动）、起卦日甲子。

1. 装卦：本卦地泽临（坤宫，世二应五），二爻丁卯木动，变地雷复。
2. 取用神：问合作看**应爻**（对方）与**世爻**（自己）关系，兼看妻财（利益）。
3. 断：世应之间生克如何、用神旺衰如何、动爻变出回头生克如何，逐项对照本篇三节给趋势判断，每条注明依据（某爻某六亲旺衰动静）。
4. 收口趋势化：**倾向可谈 / 有阻宜缓**，附应期参考与可执行提醒。一事一占，结合现实定夺。

## 七、边界与免责

- 六爻是占卜辅助参考，断的是倾向与时机，非预言；不承诺祸福成败、不打分、不吓人。
- 一事一占，同一事不反复摇卦；代占（替人问）须说明立场，取用神随之调整。
- 与八字、梅花各归各用：八字批一生格局，梅花取象快断，六爻装卦细断，不混断。测大事（婚姻、健康、大额投资）提示当参考维度，结合现实理性拍板。

## 八、出处与版本说明

- 纳甲筮法定式出汉·京房（八宫、纳甲、世应），后经《火珠林》立六亲用神之法，至清《增删卜易》《卜筮正宗》大成。本篇纳甲表、八宫世应、六亲配法、六神起法为历代通行定式，实现于 `scripts/liuyao.py` 并以定式回归测试钉死。
- 取用神分类、旺衰断法采《增删卜易》主流口径；各书于伏神、应期细节有异说，标为倾向参考。引用前先认版本，凡有争议处只作参考。
